Người dùng mới có thể đăng ký bằng email và mật khẩu. Dữ liệu tài khoản sẽ được gửi đến backend Apps Script và khi đăng nhập thành công, trang chủ chính sẽ hiện ngay trong cùng file này.

Đã gắn sẵn Apps Script Web App
Luồng hoạt động
Đăng ký → lưu vào Google Sheet → đăng nhập → hiện trang chủ chính.
Backend hiện tại
Đã dùng trực tiếp URL /exec trong file backend của bạn.
Kết nối backend
File này đã gắn sẵn Apps Script Web App URL, nên bạn có thể đăng ký và đăng nhập ngay mà không cần nhập lại link.
